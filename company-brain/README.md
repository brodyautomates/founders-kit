# Company Brain (open source)

The open-source base for a context layer. A codebase you self-host, a repo you run yourself, that ingests everything you feed it and answers questions across it with sources, building its own map of how your people, clients, and deals connect.

## What it is

- A self-hosted knowledge system (an open-source "company brain" / GBrain-style project).
- You feed it your meetings, emails, calls, chats, docs, and CRM records.
- It answers questions across all of it **with citations**, and builds a knowledge graph of your people, companies, and deals.
- It runs on its own and consolidates its memory over time.

## How capable is it

This is a real codebase, not a toy: a full repo you clone and run. Out of the box it does ingestion, synthesis with sources, and the knowledge graph. It is the base the high-end (paid) version is built on. How far you take it is up to you.

## Getting it

This is **third-party open source**. Use the upstream repository directly, don't re-host it as your own, and follow its license.

> Upstream repo: add the link to the open-source company-brain project you're using here before sharing. (Search "GBrain open source company brain" for the current project.)

## What to feed it first

1. Export your SOPs and key docs.
2. Connect email + calendar.
3. Point your note-taker at it so call transcripts land automatically.
4. Route your CRM records in.

Then ask it questions the way you'd ask a teammate who has read everything.
